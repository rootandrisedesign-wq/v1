# v1

Monorepo root: แอปพอร์ตโฟลิโอ Next.js อยู่ในโฟลเดอร์ **`portfolio/`**

## เว็บไซต์ (Production)

หลัง deploy บน Vercel ให้แก้บรรทัดด้านล่างเป็น URL จริง แล้ว commit — หรือใส่ลิงก์ที่ GitHub → **About → Website**

**Live:** _(ใส่ URL จาก Vercel เช่น `https://....vercel.app`)_

## รันบนเครื่อง

```bash
cd portfolio
npm install
npm run dev
```

## Deploy บน Vercel

1. Import repo นี้จาก GitHub
2. **Root Directory:** `portfolio`
3. Framework: Next.js (ตรวจสอบว่า `npm run build` รันที่โฟลเดอร์ดังกล่าว)
